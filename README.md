# hexo
my hexo web
优势
使用hexo+GitHub Pages 建站，免流量
绑定一级域名
使用markdown编辑 （hexo-hey）
使用终端上传和控制
在hexo官方文档下建立本地环境
git hexo hey
编辑网站设置 config.yml
打开终端，cd 到网站目录
运行 hexo g 生成静态
运行 hexo s 打开服务
修改网站内容 /admin
hexo clean
hexo g
hexo d -g
上传成功
绑定域名，两个A，一个CNAME
在github代码见CNAME，或设置里，输入刚刚CNAME的地址
完成
